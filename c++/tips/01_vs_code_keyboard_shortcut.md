#### VS Code 一些常用快捷键
> zxwang

单命令
- F1 打开命令模式
- F12 跳转到函数或变量的声明定义处
<br>

- Ctrl + x 剪切本行 或 选中内容
- Ctrl + c 复制本行 或 选中内容
- Ctrl + Enter
- Ctrl + Shift + Enter
- Ctrl + G  行跳转
- Ctrl + P  文件跳转
- Ctrl + / 注释
<br>

- Home 跳转行首
- End 跳转行末
- Ctrl + Home 跳转到文件开始
- Ctrl + End 跳转到文件末尾
<br>

- Alt + / 代码提示
- Alt + 上/下键 上下移动当前行
<br>

组合模式
- Ctrl + M, Ctrl + O 折叠所有代码

- Ctrl + M, Ctrl + L 展开所有代码

- Ctrl + K, Ctrl + T 换vscode的主题
<br>
